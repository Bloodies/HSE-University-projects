Интерфейс содержит следующие элементы:

EditText (id aValue) - значение коэффициета a.
EditText (id bValue) - значение коэффициета b.
EditText (id cValue) - значение коэффициета c.
EditText (id x1Value)- значение корня уравнения.
EditText (id x2Value) - значение корня уравнения.
TextView (id isSolutionExist) - статус количества корней:
"Any number" - любое число, возможно когда a = 0 b = 0 c = 0.
"No real roots" - нет корней, возможно когда дискриминант меньше 0 или a = 0 b = 0 c !=0.
"One root" - один корень, когда дискриминант равен 0 или a = 0 b !=0.
"Two roots" - два корня.
Учтите, что коэффициенты и корни могут быть как целые, так и вещественные.

Пользователь вводит коэффициенты квадратного уравнения: a*x^2+b*x+c=0.

После ввода всех трех коэффициентов без нажатий на какие либо кнопки производится расчет корней. Аналогичным образом, если пользователь вводит или изменяет значения корней, то автоматически пересчитываются значения коэффициентов.

Если в поля ввода введено что-либо, кроме чисел, в поле isSolutionExist необходимо вывести текст "error".

Предполагается что x1 - корень уравнения, полученный при сложении корня из дискриминанта, ﻿x2 - при вычитании.

Если уравнение вырожденное(a = 0), то поле под x2 остается пустым, но при a != 0 значения в полях корней равны друг другу

По умолчанию, если вычисляются значения коэффициентов при неизвестном, коэффициент а всегда считается равным 1.
